## Button 

Click or touch it to trigger an operation. The encapsulated logic is triggered in response to user clicks.

## Usage

```jsx
<m-button>I'm button.</m-button>
```

## API

### Props

```jsx
{
  ripple: boolean,
  raised: boolean,
  dense: boolean,
  unelevated: boolean,
  outlined: boolean,
  icon: object
}
```
